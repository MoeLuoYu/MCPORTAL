# MCPORTAL
## 让您更加简单的分享您的文件

本程序基于PHP7.2开发，理论支持所有PHP版本
### 通过测试的PHP版本： PHP-72

优点：
1.	简单易用
2.	不需要伪静态（Rewrite）支持
3.	支持高度自定义
4.	社区版完全免费，无任何版权标识

大家可以自行尝试

后台内有全面的网站设置，大家可以根据需要自行DIY

接受社区内语言更新，欢迎大家投稿

本程序依照 GNU General Public License v3.0 开源协议进行开源
